From .vsix files

You can build the vsix files from source. For superbol-vscode-platform, run the following commands:

yarn install #if not already done
drom build
make compile
yarn package

For superbol-vscode-debug, run the follwing command:

yarn compile
vsce package

With those commands you should have respectively superbol-vscode-platform.vsix and superbol-vscode-debug.vsix in the project root directory.

To install them open VSCode and go the the Extensions view.

Click on the ... at the top right of the left pane and select Install from VSIX ....

Select the superbol-vscode-debug.vsix file first to install it.

Finally select the superbol-vscode-platform.vsix file to install it.
