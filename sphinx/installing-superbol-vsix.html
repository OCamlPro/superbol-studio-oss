Installing the VSCode extension

From marketplace

Microsoft Marketplace

Download the SuperBOL Studio VSCode extension on the Visual Studio marketplace

It can also be found directly on the “extension” tab of VSCode.

OpenVSX

Download the SuperBOL Studio VSCode extension on the OpenVSX marketplace

From .vsix files

To install the extension from its VSIX file, open VSCode and go to the the Extensions view.

Click on the ⋅⋅⋅ at the top right-hand side of the left pane and select Install from VSIX….

Select the superbol-vscode-platform.vsix file to install it.
